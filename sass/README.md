### A typical top-level directory layout

```

sass
│ README.md
| main.scss
│
└───utilities/
│ │ _variables.scss
│ │ _mixins.scss
| | _functions.scss
│
└───base/
│ │ _reset.scss // Reset/normalize
│ │ _typography.scss // Typography rules
│
└───components/
│ │ _button.scss
│ │ _slider.scss
| | ...
│
└───layout/
│ │ _navigation.scss
│ │ _header.scss
| | _footer.scss
| | _sidebar.scss
| | _forms.scss
| | _grid.scss
| | ...
│
└───pages/
│ │ _home.scss
│ │ _about.scss
| | _contact.scss
│
└───themes/
│ │ _theme.scss
│ │ _admin.scss
│
└───vendors/
│ │ _bootstrap.scss
│ │ _jquery-ui.scss
|
|
```

It does not mean you have to follow everything as it is, you might not need some of them from the above structure
